# Suggested CHANGELOG Entry

## Backend API v0.2

Added:

- GUI-facing backend API under `odefit.api.backend`
- JSON-friendly serialization layer under `odefit.api.serialization`
- Single-species variable projection workflows
- Multispecies variable projection workflows
- Multistart model comparison workflows
- Bootstrap uncertainty workflows
- Profile likelihood workflows
- Identifiability diagnostics
- Backend workflow smoke-test runner
- Documentation for GUI integration and HSQC workflows

This version is intended to support GUI alpha development.
